# Task 02 – Image Generation with Pre-trained Models

## Objective
Generate images using pre-trained generative AI models.

## Technologies Used
- Python
- Google Colab
- Stable Diffusion v1.5
- Hugging Face Diffusers

## Workflow
1. Installed required libraries
2. Loaded Stable Diffusion model
3. Entered text prompts
4. Generated images
5. Saved output images

## Sample Prompts
- A futuristic smart city at sunset
- Cyberpunk city with neon lights
- Astronaut riding a horse
- AI classroom
- Robot working in office

## Output
Successfully generated images from text prompts.
